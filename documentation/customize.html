Customize to Fit your Needs

To overide the theme CSS styles you can use assets/css/custom.css for your own customization. It will be easier for your futur updates if you keep your own modifications in a separate file.

Don't forget to add custom.css in the head of your page to apply your custom CSS.

Javascript

To add javascript functions, you can use assets/js/custom.js for your own customization. This will make the future updates easier if you keep your own Javascript code seperate.

Don't forget to add custom.js in the footer of your page to apply your custom Javascript.

Create new page

To create a new page, you can use blank_page.html which give you a basic template to create your own page which template elements: blank_page.html

Page Builder

Of course, instead of use blank page template, you can use Page Builder to make your page directly from your browser, and export html.

For more information about this tool, read page builder documentation.
